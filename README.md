# MotoClub App

- Clone this repository.

```bash
 git clone https://github.com/sebasgonzac27/thesis_project_frontend.git
```

- Install dependencies.

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

- Run project.

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev
```
